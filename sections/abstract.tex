\begin{abstract}
    We introduce Dynamic Programming Supervised Learning (DPSL), a novel framework that transforms the task of training a reinforcement learning agent in a deterministic intraday trading environment into a stable and efficient supervised learning (SL) problem. The loss function of the SL agent relies on an approximation of the optimal value function that can be computed in polynomial time using dynamic programming (DP). We demonstrate that DPSL converges significantly faster and more reliably than traditional RL, even in the absence of strong predictive signals.
\end{abstract}
